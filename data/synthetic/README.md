# Synthetic Track B fixture

These files are deterministic integration fixtures generated by
`scripts/make_synthetic_fixture.py`. They exercise the feature, label, grouped
split, target, quality, calibration, no-call, and evidence contracts before real
AMRFinderPlus features are available.

They are **not biological data**. Metrics produced from them are software checks
only and must never be shown as scientific or clinical performance.

Regenerate with:

```bash
python scripts/make_synthetic_fixture.py
```
