# Real held-out results (3,000-genome K. pneumoniae cohort)

Generated by `scripts/make_pitch_results.py` on real AMRFinderPlus features.

- `pitch_metrics.csv` — per-drug metrics, grouped AND temporal splits
- `fig_calibration_real.png` — per-drug reliability curves (grouped test set)
- `fig_grouped_vs_temporal.png` — balanced accuracy across a 4-year forward gap
- `eval_grouped/`, `eval_temporal/` — full evaluate.py output
- `models/kp_grouped.joblib`, `models/kp_temporal.joblib` — trained bundles

3 genomes (573.14076, 573.26121, 573.66582) failed AMRFinderPlus annotation
(transient BV-BRC download errors) and were dropped from both splits; results
are on 2,997 genomes (grouped) / 2,813 with collection year (temporal).
