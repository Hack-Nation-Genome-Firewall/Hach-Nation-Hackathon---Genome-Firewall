01The problem

A patient is septic. The clinician must pick an antibiotic now.

Standard susceptibility testing needs a culture — 24–72 hours. Wait, and the infection advances. Guess broad-spectrum, and you may still miss the drug the bacterium already resists — while fuelling more resistance.

The genome of the isolate is available long before the culture result. Can we read resistance straight from it — safely enough to act on?

+7.6% /hr
rise in mortality per hour of ineffective therapy in septic shock (Kumar 2006)
24–72 h
culture-based AST turnaround — the window we compress to seconds
02The gap, and our fix

Today’s tools give a confident binary call. We add what makes it trustworthy.

Rule-based tools (ResFinder, CARD, AMRFinderPlus, Kleborate) report which resistance genes are present and emit resistant / susceptible — with no calibrated confidence and no way to say “I don’t know.” We keep that interpretability and add three things:

01 · CALIBRATED

Real probability

A calibrated model per drug, so a 0.9 means ~90% observed failure in held-out reality — not a raw score.

02 · THE FIREWALL

Honest abstention

An out-of-distribution detector routes unfamiliar genomes to no_call → escalate to lab instead of guessing.

03 · GLASS-BOX

Mechanism shown

Every call names the marker driving it and an evidence tier — a reason, not a black box.

On top, a stewardship layer recommends the narrowest-spectrum agent predicted to work — sparing last-resort carbapenems, safe by construction.

STEWARDSHIP → recommends ceftazidime (narrowest agent predicted to work) · confirm with lab AST
When the genome looks like nothing it trained on, it returns no_call and routes to the lab. That refusal — not the accuracy — is the point. This is the firewall.

05Results · the evidence

The confidence is real — and it holds honestly under drift.

Reliability curves sit on the diagonal — a predicted 0.9 fails ~90% of the time. Brier 0.02–0.05.

Unseen lineages: balanced accuracy 0.94–0.98. Future isolates: 0.78–0.94 — where it drops, the firewall abstains more, not guesses.

05Impact & what’s next

Genome-to-decision in seconds — with the honesty to escalate when it shouldn’t guess.

Incumbents give a confident binary call; we give a calibrated probability with an honest refusal.

2,997 K. pneumoniae genomes · BV-BRC laboratory-measured labels only (~72k computational predictions per drug explicitly excluded) · AMRFinderPlus features · calibrated logistic regression + OOD abstention + stewardship.

→ Kleborate specialist annotator → multi-species via one frozen contract → resistance-drift surveillance → combination-therapy stewardship
Safety by design. Decision support only — every prediction must be confirmed by standard laboratory AST. The system is built to abstain rather than risk a confident wrong call.
